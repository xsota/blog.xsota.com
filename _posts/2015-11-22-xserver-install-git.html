curlを入れるよ
[shell]
% wget https://github.com/bagder/curl/releases/download/curl-7_45_0/curl-7.45.0.zip
% unzip curl-7.45.0.zip
% cd curl-7.45.0
% ./configure --prefix=/home/$USER --enable-libcurl-option
% make && make install
[/shell]

gettextを入れるよ
[shell]
% wget http://ftp.gnu.org/pub/gnu/gettext/gettext-0.19.6.tar.gz
% tar zxvf gettext-0.19.6.tar.gz
% cd gettext-0.19.6
% ./configure --prefix=/home/$USER
% make && make install
[/shell]

ついにgitをどうにゅうだ〜
[shell]
% wget https://github.com/git/git/archive/master.zip
% unzip master.zip
% cd git-master
% autoconf
% ./configure --prefix=/home/$USER --with-curl=/home/$USER --with-expat
% make && make install
[/shell]

.bashrcとかに書こう
[shell]
export PATH=$HOME/bin:$PATH
[/shell]
