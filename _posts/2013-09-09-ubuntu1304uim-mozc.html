Ubuntu13.04にuim + mozcの環境を導入しました。

自宅のMacがGoogle日本語入力なのでubuntuでもそれに近い作業感出したかった。

以下メモ


# apt-get install ibus-mozc mozc-server mozc-utils-gui
# apt-get install uim uim-mozc uim-utils

システム設定>言語サポート>キーボード入力に使うIMシステム>uim

その後


$ uim-toolbar-gtk

uimの設定を開き変換キーと無変換キーでオンオフ切り替えできるようにする。

これでMac風だ。やったね。
