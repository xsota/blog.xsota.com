1.JDK

ここから
http://www.oracle.com/technetwork/java/javase/downloads/jdk7-downloads-1880260.html
自分の環境に合ったJDKをdownload
今回はVine Linux 6.1の入った32bitマシンにインストールするので
” jdk-7u21-linux-i586.rpm”を選択します。

ダウンロードが終わったら端末上で
 % sudo rpm --install jdk-7u21-linux-i586.rpm
を実行します。


画像の用に表示されたら次に進む。

2.Eclipse&Android SDK

http://developer.android.com/sdk/index.html
から自分の環境にあったADT bundleをダウンロード。
今回は 1と同じくLinux 32-bitをダウンロード。

 

$HOMEかどっかに展開します。

 次にeclipseを起動します。

上記画像のようなロゴがでてきたら成功です。

起動したらWindow > Preferences >Androidを開き、SDK Location というところを確認してください。SDKの場所が指定されていれば次に進みます。
されていなかった場合は自分でSDKの場所を指定。

パスを通したら Help > Install New Software を開きWork Withに
 http://dl-ssl.google.com/android/eclipse/site.xml  を追加する

とりあえずSelect All で全部インストール。

次にWindow > Android SDK Managerを開きます。
画像のようなウィンドウがでてくるので、必要なものをインストールしてください。

以下のエラーがでてきたら。
Failed to fetch URL https://dl-ssl.google.com/android/repository/repository-7.xml …, reason: SSLPeerUnverified peer not authenticated

 Window > Android SDK Manager > Tools > Options > Force https://… sources to be fetched using http://… にチェックを入れると怒られなくなるかもしれません。
