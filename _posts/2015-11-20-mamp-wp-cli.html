wp-cliをダウンロードするよ〜

[shell]
% cd /Applications/MAMP/Library/bin
% curl -O https://raw.githubusercontent.com/wp-cli/builds/gh-pages/phar/wp-cli.phar #wp-cliをダウンロード
% php wp-cli.phar --info #wp-cliが動いてるかinfoみてみるよ〜
% chmod +x wp-cli.phar #実行できるようにしとく
% mv wp-cli.phar wp #名前をwpに変えておく(wpじゃなくてもいいよ)
[/shell]

.zshrcとかに書いたよ。

phpのversionのところは適宜変更してね。
[shell]
export MAMP=/Applications/MAMP/bin/php/php5.5.14/bin:/Applications/MAMP/Library/bin:
export PATH=$MAMP:$PATH
[/shell]

よみこみなおしたよ

[shell]
% source ~/.zshrc
[/shell]

おしまい

[shell]
% wp help
[/shell]
